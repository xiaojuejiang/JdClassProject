# 前言

这是一个类电商的小项目,主要用来个人练手用的。其中，包含了前后端的所有功能，由于没有服务器，只是在本地进行测试使用。

## 技术栈

node + vue + router + vue-router+ mui +  zepto  +  less + ES6/7
